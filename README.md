# Zomato Restaurant Reviews – Data Science & AI Mini Project

## Domain
Data Science and Artificial Intelligence (NLP – Basic)

## Project Overview
This project focuses on analyzing Zomato restaurant data using Data Science techniques and basic AI concepts.  
The dataset includes restaurant metadata and customer reviews, which are used to extract meaningful insights about restaurant performance and customer sentiment.

This is a small-scale project developed as part of the Labmentix Internship assignment.

---

## Datasets Used
1. **Zomato Restaurant names and Metadata.csv**
   - Contains restaurant details such as name, location, ratings, and other metadata.

2. **Zomato Restaurant reviews.csv**
   - Contains customer reviews used for text analysis and sentiment understanding.

---

## Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Analyze restaurant ratings and review patterns
- Apply basic AI/NLP techniques on review text
- Extract business-relevant insights

---

## Tools & Technologies
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook
- Basic NLP (Sentiment Analysis / Text Processing)
